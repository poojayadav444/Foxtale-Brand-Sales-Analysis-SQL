# Foxtale Brand Sales Analysis using SQL

Project Overview
This project analyzes Foxtale skincare brand sales data using SQL to understand
revenue trends, product performance, customer behavior, and category-wise sales.
The dataset used is a sample (mock) dataset created to simulate real-world
e-commerce transactions.

Objective
 Analyze total sales revenue
 Identify top-performing Foxtale products
 Understand category-wise sales performance
 Analyze monthly sales trends
 Identify high-value customers

Tools Used
 MySQL
 SQL

Database Structure
The project consists of four relational tables:
 products
 customers
 orders
 order_details

Analysis Performed
 Total revenue calculation using quantity and price
 Product-wise revenue analysis
 Category-wise sales analysis
 Monthly sales trend analysis
 Customer-wise spending analysis

Key Insights
 Serum and Treatment categories generate high revenue
 Moisturizers and Sunscreens show consistent demand
 Repeat customers contribute significantly to total sales
 Sales show a positive trend across multiple months
